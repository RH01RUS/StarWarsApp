# Star Wars HoloNet - Приложение для просмотра вселенной Star Wars

Современное Android-приложение с голографическим интерфейсом, офлайн-режимом и системой сравнения персонажей.

## 📱 О приложении

Star Wars HoloNet - это современное Android-приложение, которое предоставляет доступ к базе данных Star Wars API (SWAPI). Приложение позволяет просматривать информацию о персонажах, планетах, фильмах, звездолетах, транспортных средствах и видах из вселенной Star Wars.

## 🎨 Нестандартные дизайн-решения

### 1. Голографический интерфейс в стиле Star Wars
- **Тёмная тема с градиентом**
- **Золотые акценты**: Все активные элементы подсвечены золотым цветом (#FFD700), что отсылает к классическому логотипу Star Wars
- **Красные акценты**: Кнопка сравнения и индикаторы загрузки выполнены в цвете светового меча ситхов (#E94560)
- **Анимированные звёзды**: Фон детальных экранов содержит мерцающие звёзды, создающие эффект нахождения в космосе

### 2. Анимированные карточки персонажей
- **Анимация появления**: Карточки появляются с эффектом затухания и скольжения, с задержкой в зависимости от индекса (каждая следующая карточка появляется на 50 мс позже)
- **Эффект нажатия**: При нажатии карточка слегка уменьшается (scale 0.98), создавая тактильный отклик
- **Градиентный фон**: Каждая карточка имеет горизонтальный градиент с красным оттенком, имитирующий голографическую проекцию

### 3. Интеллектуальное кэширование
- **Двойное кэширование**: Данные хранятся одновременно в памяти и в Room Database
- **Умный fallback**: При недоступности API (таймаут 5 секунд) автоматически подгружаются кэшированные данные
- **Персистентное хранение**: Данные сохраняются на диске и доступны офлайн даже после перезагрузки приложения
- **Кэширование всех категорий**: В отличие от базовой реализации, кэшируются не только персонажи, но и все 6 категорий

### 4. Уникальная система сравнения персонажей
- **Выбор нескольких персонажей**: Кнопка "+" на карточке позволяет добавлять персонажей в список сравнения (максимум 4)
- **Визуальная индикация**: Выбранные персонажи отмечаются зелёной галочкой
- **Кнопка VS**: Появляется в верхней панели только когда выбрано 2+ персонажа
- **Детальное сравнение**: Экран сравнения показывает все характеристики выбранных персонажей в табличном виде

### 5. Анимированный Splash Screen
- **Пульсирующая звезда**: Анимация масштабирования звезды с 0.8 до 1.2
- **Мерцающий текст**: Текст "STAR WARS" и "HOLONET" пульсируют с изменением прозрачности
- **Анимированные точки загрузки**: Три точки последовательно меняют прозрачность
- **Автоматический переход**: Через 3 секунды автоматический переход в приложение

### 6. Оптимизированная работа с API
- **Таймауты запросов**: Все сетевые запросы имеют таймаут 3-5 секунд для предотвращения зависаний
- **Ленивая загрузка**: Использование LazyColumn для эффективной работы с большими списками

### 7. Единый компонент для всех категорий
- **UnifiedCard**: Создан универсальный компонент карточки, используемый для всех 6 категорий
- **Адаптивные иконки**: Для каждой категории используется своя эмодзи-иконка 
- **Единый стиль**: Все экраны приложения выдержаны в единой стилистике
- **Fallback на кэш**: При любой ошибке сети автоматически подгружаются кэшированные данные

## 🛠 Технологический стек

| Технология | Назначение |
|------------|------------|
| **Kotlin** | Основной язык разработки |
| **Jetpack Compose** | Декларативный UI |
| **Room** | Локальное кэширование данных |
| **Retrofit + OkHttp** | HTTP-запросы к SWAPI |
| **Hilt** | Внедрение зависимостей |
| **Coroutines + Flow** | Асинхронность и потоки данных |
| **Navigation Compose** | Навигация между экранами |


## 🚀 Уникальные особенности

1. **6 полноценных категорий** вместо базовых 2 (добавлены Films, Starships, Vehicles, Species)
2. **Полнотекстовый поиск** по всем категориям
3. **Система сравнения персонажей** с визуальным выбором
4. **Офлайн-режим** с кэшированием всех данных
5. **Анимированный онбординг** при первом запуске
6. **Splash Screen** в стиле Star Wars
